# KaBaActivitys
### Connecting people through their interest

### Installs needed
Frontend: 
- Node.js
    - (https://nodejs.org/en/)
- Angular
    - npm install -g @angular/cli
- Ionic
    - npm install -g ionic
    
Backend
- Java
- Docker 
    - Windows: (https://docs.docker.com/docker-for-windows/install/)
    
## Getting started
Start Frontend:
- ionic serve

Start Docker:
- docker start "image name"

Start Backend
- Start via IntelliJ
